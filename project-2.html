Juan Translator App

Juan: A Voice Recognition Language Translator Application was designed and coded in Android Studio. The application translates between English and eight different Filipino dialects

Project Overview

The Juan Translator App is a feature-rich language translation application that enables seamless communication between English and eight Filipino dialects. With its user-friendly interface, real-time translation capabilities, and reliable performance, Juan Translator stands as a valuable tool for bridging language gaps and fostering effective cross-cultural communication on Android devices.

Key Features: Text Translator, Voice Translation, Language Map and Basic Expression

Tools Used

Android Studio
Java
GUI Design
Google Voice Recognition API
